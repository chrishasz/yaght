## Contributed properties

### Page Level

The following properties can be individually set on each page.

#### page.hideBreadcrumb

*default: false*
Set this to true to prevent the breadcrumb control from rendering on this page.

#### header_pages

``` yml
header_pages:
  - name: about.md
    title: About Page
    url: /about
  - name: about2.md
    title: About Page 2
    url: /about2
```
